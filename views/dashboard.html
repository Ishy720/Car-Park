User Data Schema

Here you can control the data the system holds about the users registered to the website. You can add and remove new fields in the format {"[DATA_FIELD_NAME]:[TYPE]"}.

Examples:

"middleName":"" to add a word (string) value called middleName with the default value of nothing.

"isStudent":false to add a true/false state called isStudent with the default value of false.

"discountPercentage":15 to add a numerical value called discountPercentage with the default value of 15.

Please ensure to separate each data field with a comma.
